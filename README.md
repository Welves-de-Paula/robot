    # Robô 4WD com Controle WiFi

## Descrição

Projeto de robô carrinho 4WD (tração nas 4 rodas) com controle remoto via WiFi. O sistema permite controlar o movimento do robô através de uma interface web conectada à rede WiFi.

## Objetivo

Criar um robô controlado remotamente via WiFi, utilizando Arduino com módulo ESP8266 para comunicação wireless e motor shield L293D para controle dos motores.

## Componentes Necessários

### Hardware

- 1x Placa UNO WiFi ATmega328P + ESP8266
- 1x L293D – Drive Ponte H (Motor Shield)
- 2x Baterias de Íon de Lítio 3.7V 500mAh
- 4x Motor DC 3-6V com Caixa de Redução e Eixo Duplo
- 4x Roda de Plástico 65mm

## Funcionalidades

- Controle remoto via WiFi
- Interface web para comando do robô
- Tração 4WD para melhor mobilidade
- Comunicação wireless através do módulo ESP8266
